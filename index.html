搭建Hexo博客
付出行动今天来谈谈，写博客。说起写博客，其实自己写博客的时间也不长，也就几个月时间，之前学习的时候也会经常看一些大佬写博客，当时觉得奇怪，就觉得写这个东西有点浪费时间，自己知道好的技术或者一些解决思路。等写一段时间博客会慢慢发现我们是在主动学习，非常的有效率。同时在写的过程中发现，很多细节没琢磨明白，但为了把事情讲全面，需要查阅很多资料也是为了准确无误（害怕误人子弟，不招致谩骂）使得对知识的应用有了深刻的理解。写博客的益处不仅更好的保留知识，是一个更好的传递力量，也会让我们认识更多的同道中人，所以今天付出行动。利用hexo + gitHub page 从0到1搭建一个属于自己的博客。 准备工
